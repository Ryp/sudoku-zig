# Sudoku

## Building

This should get you going after cloning the repo:
```sh
$ zig build run
```

## Controls

| Action         | Key                |
|----------------|--------------------|
| Quit game      | Escape             |
| Select cell    | Left mouse button  |
| Move selection | Arrow keys         |
| Place number   | <1-9,A-G>          |
| Clear number   | <0>, Del           |
| Toggle guess   | Shift + <1-9,A-G>  |
| Undo           | Ctrl + Z           |
| Redo           | Ctrl + Shift + Z   |
| Fill guesses   | H                  |
| Clear guesses  | Shift + H          |
